## Local development

Install dependencies:

```
npm install
cd functions
npm install
cd ..
```

You will also need to have Java installed to run the emulators.

Do an initial build of the functions:

```
cd functions
npm run build
cd ..
```

Then run the emulators:

```
npm run emulators
```

Go to http://localhost:5000 and enjoy!
